# Raw Data from the analyses

You can browse the following folders:

1. [Bound_ions_analysis](Bound_ions_analysis): Data for the average number of bound divalent ions and for the stable bound divalent ions.
2. [DCCM_analysis](DCCM_analysis): Dynamical cross-correlation matrices of nucleosomal DNA.
3. [DNA_geometrical_parmerater](DNA_geometrical_parmerater): DNA geometrical parameters (base pair and base pair step parmeraters) of per base pair for all frames 
4. [Electrostatic_force_calculation](Electrostatic_force_calculation): Electrostatic force projections on DNA at different SHLs.
5. [Gaping_distance](Gaping_distance):  Gaping distance at different SHLs for nucleosome with and without tails.
6. [MMPBSA_calculation](MMPBSA_calculation): Histone-DNA binding free energies calculated using MM/GBSA approach.
7. [Radius_of_gyration](Radius_of_gyration): Radius of gyration of tailless nucleosomes.
8. [RDF_analysis](RDF_analysis): Radial distribution functions of cations for different simulation sets.
9. [Residence_time](Residence_time): Full tail residence time on DNA.
10. [Histone_DNA_contacts](Histone_DNA_contacts):  Histone DNA contacts per base pair and per tail residue.
11. [Unwrapped_base_pairs](Unwrapped_base_pairs): Number of unwrapped base pairs in tailless nucleosomes.
12. [Youngs_modulus](Youngs_modulus): Young's modulus of nucleosome with and without tails.
